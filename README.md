# Miarisa - Precision Agronomy Platform

Landing page/presentation site for Miarisa — an AI-powered Precision Agronomy platform for industrial apple orchards.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animation:** Framer Motion

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design System

### Colors
- **Primary:** `#1a365d` (Deep Royal Blue)
- **Accent Teal:** `#00897b` (Laser Teal)
- **Accent Red:** `#dc2626` (Critical Alerts)
- **Background:** `#f8fafc` (Very light cool grey)

### Typography
- **Font:** Inter / Helvetica Neue
- Clean, sans-serif, highly readable
